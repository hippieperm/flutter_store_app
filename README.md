# Flutter Store App 🛍️

모던하고 세련된 쇼핑몰 앱을 Flutter로 구현한 프로젝트입니다.

## ✨ 주요 기능

- 여성, 키즈, 신발, 가방 카테고리 분류
- 직관적인 UI/UX 디자인
- 반응형 레이아웃

## 🛠 기술 스택

- Flutter SDK 3.5.4
- Dart
- Material Design

## 🚀 시작하기

### 필수 조건

- Flutter SDK 3.5.4 이상
- Dart SDK
- Android Studio / VS Code
- Git

### 설치 방법

1. 저장소 클론

```bash
git clone https://github.com/yourusername/flutter_store_app.git
```

2. 프로젝트 디렉토리로 이동

```bash
cd flutter_store_app
```

3. 의존성 패키지 설치

```bash
flutter pub get
```

4. 앱 실행

```bash
flutter run
```
